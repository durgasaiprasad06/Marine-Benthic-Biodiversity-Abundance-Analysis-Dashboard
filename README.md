# 🌊 Marine Benthic Biodiversity & Abundance Analysis

This project presents an **interactive Power BI dashboard** designed to analyze **marine benthic biodiversity**, focusing on species diversity, abundance patterns, taxonomic classification, and site-wise distribution.

---

## 📊 Project Overview

Marine benthic ecosystems consist of organisms living on or near the ocean floor. This dashboard helps in understanding biodiversity patterns and ecological distribution.

---

## 🚀 Dashboard Features

### 🔹 Overview Analysis

* Total Species: **685**
* Total Abundance: **~411,000**
* Phylum-wise contribution (Donut Chart)
* Depth vs Abundance (Scatter Plot)
* Species-wise distribution (Bar Chart)

---

### 🔹 Taxonomic Classification

* Genus filter (Slicer)
* Genus count & Average Aphia ID by Class
* Authority-wise genus contribution (Waterfall Chart)
* Taxonomic flow (Sankey Diagram)

---

### 🔹 Species & Site Analysis

* Top Site: **Denny Way (~124K)**
* Site-wise abundance table
* Genus distribution (Treemap)

---

## 🛠️ Tools Used

* Power BI
* Power Query
* DAX

---

## 📂 Dataset

Includes species, genus, class, phylum, abundance, site, depth, and taxonomic identifiers.
Dataset Link:-   https://catalog.data.gov/dataset/marine-benthic-abundance-data

---

## 📌 Key Insights

* Annelida is the dominant phylum
* Higher abundance at lower depths
* Denny Way is the most abundant site
